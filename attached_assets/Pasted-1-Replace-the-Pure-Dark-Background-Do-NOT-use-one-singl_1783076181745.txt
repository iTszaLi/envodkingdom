1. Replace the Pure Dark Background

Do NOT use one single dark background for the whole website.

Instead create alternating premium sections.

Example flow:

Hero
→ deep navy gradient

Services
→ light gray (#F8FAFC)

Why Choose Us
→ white

Customs Clearance
→ navy gradient

Industries
→ white

Global Network
→ very light gray

Testimonials
→ white

CTA
→ premium navy

Footer
→ dark navy

This creates visual rhythm and prevents eye fatigue.

2. Remove the Visible Grid Background

The current grid makes the website feel like a software dashboard.

Either:

• remove it completely

OR

make it only 3-5% opacity so users barely notice it.

The focus should be on the content—not the background.

3. Add Premium Section Backgrounds

Instead of plain colors, use subtle backgrounds such as:

• soft gradients

• blurred radial lights

• abstract logistics patterns

• world map watermark

• container silhouettes

• cargo route lines

• shipping route dots

These should stay around 5–8% opacity.

4. Use Full Width Images Between Sections

Break up text-heavy sections.

Between sections insert cinematic full-width logistics imagery:

• Container terminal

• Cargo vessel

• Aircraft loading

• Customs inspection

• Warehouse automation

• Project cargo cranes

• GCC trucking convoy

Overlay each image with a dark navy gradient so text remains readable.

5. Give Every Section Its Own Personality

Avoid repeating the same design.

Examples:

Services
→ light background

Stats
→ navy with animated counters

Industries
→ white with large images

Timeline
→ light gray

Testimonials
→ premium glass effect

CTA
→ bold gradient

Every section should immediately feel different.

7. Premium Card Design

On light backgrounds:

• white cards

• subtle shadows

• 14–18px border radius

• soft borders

• hover elevation

• image header

• colored service icon

Cards should resemble Apple, Stripe, DHL, Maersk, or DP World websites—not dashboard widgets.

8. Add More White Space

Increase vertical spacing significantly.

Each section should have around:

120–180px top padding

120–180px bottom padding

More breathing room creates a premium appearance.

9. Improve Typography

Increase heading sizes.

Use larger spacing between headings and paragraphs.

Limit text width for better readability.

Highlight keywords in ENVOD red.

10. Premium Background Animation

Add subtle motion:

• slow moving gradient

• floating light blobs

• moving world map glow

• cargo route animations

Keep everything extremely subtle (2–5 seconds).

Avoid flashy animations.

11. Make CTAs Stand Out

Current buttons blend into the page.

Use:

• red primary buttons

• white secondary buttons

• hover lift

• shadow

• animated arrow

• smooth transitions

The "Get Quote" CTA should instantly catch the visitor's eye.

12. Enterprise Look

The final design should resemble the quality of:

• Maersk

• DP World

• DHL

• FedEx

• Kuehne + Nagel

• Stripe

• Apple

• AWS

The website should immediately communicate:

Global logistics expertise
High-value project cargo capability
Fast customs clearance
Premium enterprise service
Trust and professionalism

Avoid making it look like a SaaS dashboard or developer portfolio. The overall impression should be that of a world-class logistics company trusted by government agencies, multinational corporations, and major industrial clients.
